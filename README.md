# SILLY Interpreter v0.6
Interprets Simple Interpreted Language Like Yes (SILLY) programs.

## How to Use
Make a SILLY program and interpret it like this:
```
py src/main.py <file_path>
```
(this assumes you're in the repo root)

For example:
```
py src/main.py tests/test.sl
```
If you don't know how to write SILLY programs, read the documentation.
